mcp-server.md

Read-only MCP (phase 1)

stdio MCP over SQLite: operators inspect signals, outcomes, rewards, exchanges, and payments without hand-running SQL. HTTP server stays stdlib-only; MCP lives in the mcp uv group.

Tool prefix xchange_* Purpose
xchange_list_support_signals Exception queue, filters, pagination (limit ≤ 100).
xchange_get_outcome_summary Counts by reward state; optional student_id.
xchange_get_reward_state Ledger + evidence list + payment summaries for one reward.
xchange_list_exchange_requests Token exchange requests with filters.
xchange_list_payment_confirmations Payment rows without raw Stripe JSON (operator-safe).
Non-goals (v0): No MCP write tools (resolve, ingest, Stripe). HMAC webhook remains HTTP-only.
