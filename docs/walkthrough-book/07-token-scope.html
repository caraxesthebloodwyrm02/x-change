RewardToken scope

Scope emerges from usage; no closed “scope enum” that must be extended preemptively.

Token is

  • x-change governance unit on the ledger.
  • Epistemic credential tied to cognitive / educational outcome.
  • Issued per reward (v0 default: one token per reward).
  • Immutable after issuance (rarity_score stamped).
  • Certificate + evidence anchor + optional exchange into value scopes.
  • InsightTier captures epistemic depth.

Token is not

  • Not external currency or money exchange.
  • Not a Glass/GRID signal or rate-limit token.
  • Not an API license or quota primitive.
