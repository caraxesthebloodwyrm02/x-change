Lifecycle states (v0)

Only the policy layer advances state. Raw evidence and Stripe JSON propose transitions; they never bypass validation.

drafted Intention; not yet earned.
→
earned Evidence satisfies PSC.
→
payment_pending Awaiting Stripe match.
→
payment_confirmed Webhook + metadata OK.
→
student_acknowledged Explicit ack recorded.
review_requested — May be entered from any state when student or agent flags a mismatch.

RewardToken

Governance unit on the ledger (reward_token_amount); not ecosystem signal-rate tokens. Prior Cascade token work stays reference_only unless promoted here.

Student outcome

outcome_state: unknown → delivered_pending_ack → acknowledged / review_open. Stripe proves funds movement, not learning outcome.
