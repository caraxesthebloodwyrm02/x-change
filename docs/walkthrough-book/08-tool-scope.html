Tools as evidence sources

A tool is an observable operation: stateful, traceable, auditable, and non-inferable for policy booleans.

Source Evidence Notes
Glass glass_session_event, failure_snapshot Provenance glass_ingest; no automatic transition without explicit flags.
GRID agent_interpretation (provisional) Substantiation payload; provenance grid_substantiation.
Calculator — Consumer of RewardToken data, not an evidence source.

Tools propose transitions via ingest_bool() fields such as contract_satisfied and ready_for_payment; tool state alone must not imply reward state.
